# What is it for?
Simple script to replace Deepin's default file manager with the latest version of Nautilus

# How to

1. Download the script
2. Open the terminal
3. Add the execution permissions
```shell
  sudo chmod u+x script.sh
```
4. Run script
```shell
  ./script.sh
```
5. Enjoy :)
